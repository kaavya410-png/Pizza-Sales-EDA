Pizza Sales Data Analysis using EDA

Overview

This project analyzes a pizza restaurant's sales dataset using **Exploratory Data Analysis (EDA)** to identify the factors influencing the best-selling and least-selling pizzas. A **Linear Regression** model was used to validate the relationship between pizza attributes and sales quantity.


Objectives

- Identify best and least-selling pizzas.
- Analyze the impact of price, size, and category on sales.
- Perform ingredient frequency analysis.
- Provide business recommendations based on data insights.

Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

Key Analysis

- Data Cleaning & Preprocessing
- Sales Trend Analysis
- Best & Least-Selling Pizza Identification
- Feature Influence Analysis
- Ingredient Frequency Analysis
- Business Recommendations

Key Findings

- **The Big Meat Pizza (Small)** was the best-selling pizza.
- **The Greek Pizza (XXL)** was the least-selling pizza.
- Higher prices and larger pizza sizes were associated with lower sales.
- Popular ingredients included **Garlic, Tomatoes, Mozzarella Cheese, and Pepperoni**.
